ACE

Arbitrary Code Execution, or ACE for short, is doable on every version of Pokémon FireRed/LeafGreen using the mail glitch. Yes, even on the Switch. See Papa Jefe's video "FireRed and LeafGreen glitches are basically DLC" and the resources linked in its description for more details.

Note: The ACE codes may differ between languages, versions, and old emulators vs new emulators/real hardware. Please make sure you are using the correct codes, or you might break your game.
